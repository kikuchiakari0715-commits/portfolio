Skill

Tools and skills I use to create.

制作に使用しているスキルとツール

HTML5/CSS3 実務経験あり

HTML5とCSS3を使用して、Webサイトのコーディングが可能です。レスポンシブデザインにも対応し、見やすく使いやすいレイアウトを意識して制作しています。

WordPress実務経験あり

WordPressを使用したサイト制作の基本操作が可能です。テーマのカスタマイズや記事投稿など、運用を意識したサイト作りを行っています。

JavaScript

JavaScriptを使い、簡単な動きやインタラクションの実装が可能です。ユーザーにとって心地よい動きを取り入れたWebページ制作を心がけています。

Photoshop実務経験あり

Photoshopを使用した画像編集やバナー制作が可能です。色調補正やレタッチなど、Web掲載に適した画像加工を行っています。

Illustrator実務経験あり

Illustratorを使用して、ロゴや簡単なイラスト、アイコン制作が可能です。デザインの統一感を意識した制作を行っています。

Figma

Figmaを使用してワイヤーフレームやデザインカンプの制作が可能です。チームでの共有やフィードバックを意識したデザイン作成を行っています。
